# Aprecis — Landing

Cinematic luxury hero for Aprecis. Vite + React + TypeScript + Tailwind CSS v4 + Lucide.

```bash
cd landing
npm install
npm run dev      # local dev server
npm run build    # production build -> dist/
```

The hero background video lives at `public/hero.mp4`.

## Structure
- `src/components/Hero.tsx` — full-screen 100svh video hero, overlays, copy, CTAs
- `src/components/Navbar.tsx` — glass navbar, scroll-aware, animated mobile sheet
- `src/index.css` — Tailwind theme tokens, fonts, reveal animations
